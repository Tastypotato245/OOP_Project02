## Build and Run
### Build
```bash
$ make
```
### Run
```bash
$ ./inf_int 9999999999999999999 + 999999
```
```bash
$ ./inf_int 12345678901234567890 \* 123456789
```
```bash
$ ./inf_int
Input: 54 * 123456788888
Output: 6666666599952
Input: 2345234523525345 + 22233322
Output: 2345234545758667
Input: ^C
```
